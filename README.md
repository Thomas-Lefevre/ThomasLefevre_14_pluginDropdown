# React Dropdown

## Install

```bash
npm install --save pluginDropdown
```

## Example

```jsx
import React from 'react';
import Dropdown from "react-ddown";
import 'react-ddown/src/Dropdown.css';

function App() {
    return (
        <Dropdown 
            list={[datas]}
            setValue={func}
        />
    )
}

export default App
```
